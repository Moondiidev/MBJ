Ажилаа бүтээмээр байн уу?

Асуух асуулт байна гэж үү? Бидэнтэй ямарч цагт холбогдож бодит хүмүүсээс шуурхай тусламж авнуу.

Чөлөөт ажиллагсадад төлбөр зөвхөн өөрийг чинь авсан үйлчилгээндээ сэтгэл хангалуун гэсэн үед л олгогддог.

Хэр их мөнгө төлөхөө урдчилан мэд. Цагийн цалин гэж байхгуй, ганц удаа төлж сонгосон үйлчилгээгээ хүрт.
